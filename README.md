# pNotes
pNotes is just a very simple open-source notepad written using Python with Tkinter library.

---

To use the source code install the Tkinter (in case it is not installed) and than you can just simply run it.

### Linux

```
sudo apt install python3-tk
```

### macOS

```
cd ~/pNotes
python3 -m venv .venv
.venv/bin/pip install tk
```

### on Windows Tkinter always already pre-installed
